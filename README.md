# 🌍 Site Oficial - Asociația Sindromul Morquio

> **Slogan:** Împreună și informați suntem mai puternici!

Acesta este codul sursă al platformei web pentru **Asociația Sindromul Morquio**. Proiectul este construit cu **Nuxt 3**, **Vuetify 3** și **Nuxt Content**, fiind optimizat pentru viteză, accesibilitate și transparență.

## 🎯 Misiunea Noastră
Asociația luptă pentru dreptul la o viață normală pentru pacienții cu **boli genetice rare (în special Sindromul Morquio)**, investind în sănătate, recuperare și integrarea copiilor defavorizați în școlile de masă.

## 💻 Transparență Digitală (Open Source)
Am ales să facem codul acestui site public pentru a ajuta și alte asociații mici care au nevoie de o infrastructură digitală solidă, fără costuri de dezvoltare.

⚠️ **Important:** Dacă doriți să folosiți acest cod sau aveți nevoie de ajutor pentru implementare, **vă rugăm să ne contactați în prealabil** la `contact@asociatiamorquio.ro`. Oferim sprijin și îndrumare colegială.

## 🛠️ Tehnologii folosite
* **Framework:** [Nuxt 3](https://nuxt.com/)
* **UI Library:** [Vuetify 3](https://vuetifyjs.com/)
* **Content:** [Nuxt Content v2](https://content.nuxtjs.org/) (Markdown based)
* **Hosting:** Optimizat pentru medii statice (Netlify, Vercel, GitHub Pages)
